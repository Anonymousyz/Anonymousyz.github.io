01 / 长期主题

从研究到决策

AI 可以很快生成一份完整答案。真正困难的部分，是确认问题有没有问对、结论是否站得住，以及下一步该怎样验证。

01

真正要决定什么？

研究不从搜索开始。先把宽泛话题改成有决策者、有时间限制、也有现实约束的问题。

02

结论凭什么成立？

把事实、判断和假设分开，同时检查来源是否适用，并主动寻找反面证据。

03

下一步怎样验证？

把建议写成真实选项，说明适用条件、责任人，以及什么情况应该停止或改型。

03 / 验证状态

哪些已经验证，哪些还没有

Release、测试和合成案例可以证明工具确实存在并能运行，但不能证明它已经被市场采用。

已验证

发布物可以公开检查

  • 三个仓库均有正式公开 Release 和可下载资产。
  • 工具仓库提供测试、合成案例或在线链接核验。
  • 使用边界、许可证和反馈入口可以在仓库中查看。
尚未验证

外部采用仍缺少独立证据

  • 目前没有依据声称工具已被广泛采用或形成稳定用户群。
  • 陌生用户复现、外部 Issue 和独立引用仍然不足。
  • “从研究到决策”文章系列尚未形成成熟的公开连载。

目前正在征集首批外部试用和复现。可以通过 GitHub Issue 反馈，但请不要上传敏感或未经授权的材料。

04 / 关于我

产业研究，是我理解问题的起点

北京大学硕士，产业研究与咨询从业者。我的工作长期涉及政策、产业趋势、数字化转型和复杂决策材料。

AI 治理是我持续积累的专业方向。我不拿头衔替代项目、作品和外部检验。
