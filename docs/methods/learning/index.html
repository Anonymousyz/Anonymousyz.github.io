持续学习 · 00

持续学习好奇心 · 目标 · 检验 · 调整

我喜欢弄清新问题，也愿意为此花时间。开始学习时，我先确定这次要会解释什么、判断什么或做成什么，然后选材料、做练习；最后用写作、讲解或实际任务检查自己是否真懂。

好奇心把我带到新问题前

遇到一个判断，我常会追问它为什么成立；看到一个系统，也想弄清它为什么这样运行。问题会越查越多，所以我会按手头工作的轻重缓急安排学习。

开始前，我会写下这次要解释的现象、要作的判断或要完成的东西。目标写得越具体，越容易选择材料，也越容易看出自己学到了什么。

新概念要和已有知识接上

我会先用综述或入门材料找到概念和争议，再回到标准、论文、政策文件或产品文档核对关键定义。不同来源说法不一时，先查时间、对象、定义、统计口径和适用条件。

新概念要接到正在处理的问题上：它和哪些旧概念有关，能解释什么，到了什么条件会失效。只记住一个术语，换到别的问题上往往还是不会用。

学习方法只在需要时使用

我会合上材料，试着把关键内容说出来。这在学习研究中叫“提取练习”。如果只能重复术语，不能用自己的话解释，就说明还没有真正弄懂。需要长期记住的内容，我也会分隔一段时间再复习。

这些方法按需要选用。用了多少方法、读了多少材料都不是结果，最后仍要看能否写清、讲明或把事情做出来。

写出来，才能看出是否真懂

我会把所学内容写成一段说明，画出概念之间的关系，讲给别人听，或做成一条能运行的规则。写不清、讲不明、做不出来，通常是定义还含混、推理有跳步或材料不够。

遇到概念说不清、公式写不全、材料撑不起结论或程序与规则对不上，我就回到来源重查。判断有没有学会，要看当前工作能不能完成，不能只数读过多少材料。

换个问题，检验能不能用

换一组材料，仍能按同一口径判断，属于近迁移；换到另一个领域，还能认出相同结构，属于远迁移。后一种情况尤其要重新核对机制和条件，不能拿类比当证据。

实际使用时出了问题，我会分清是知识有缺口、当时想不起来、场景判断错了，还是方法本身不合适，再决定补材料、换练习或重新梳理概念。

根据结果调整学习方法

熟悉一段话不等于掌握。我会记下自己原来怎么理解、在哪里卡住、哪次写作或应用暴露了问题，以及后来为什么改。心理学把这类自我检查称为“元认知监测”。

如果写作或实际应用没有改善，我会换一种学法：合上材料回忆，找反例，请别人听我解释，或把任务拆小后再练。

说明

适用边界

  • 好奇心也会让人分心。任务有明确期限时，我先学习完成当前工作所必需的内容。
  • 能够较快了解一个领域，不等于取得了专业资格。涉及专业结论，仍要请相应人员复核。
  • 测评和学习方法只能说明偏好与过程。能否做好工作，要看作品、项目结果和长期表现。
